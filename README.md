# ConnectaTel_Analysis
Análisis exploratorio de datos para identificar patrones de uso en servicios de telecomunicaciones
📱 Análisis de Comportamiento de Usuarios - ConnectaTel
Análisis exploratorio de datos para identificar patrones de uso en servicios de telecomunicaciones

🎯 Descripción del Proyecto
Este proyecto analiza el comportamiento de usuarios de ConnectaTel, una empresa de telecomunicaciones con operaciones en México y Colombia. El objetivo es identificar patrones de uso, detectar comportamientos atípicos y segmentar clientes para optimizar la oferta comercial.

📊 Datasets Utilizados
plans.csv: Catálogo de planes (precios, minutos, GB incluidos)
users_latam.csv: Información demográfica de clientes
usage.csv: Datos de uso real (llamadas y mensajes)
🔍 Análisis Realizado
Limpieza y validación de datos de múltiples fuentes
Análisis estadístico descriptivo de patrones de uso
Detección de outliers mediante métodos estadísticos
Segmentación de usuarios por edad y comportamiento de uso
Visualizaciones para identificar insights comerciales
🛠️ Tecnologías
Python • Pandas • NumPy • Matplotlib • Seaborn • Jupyter Notebook

📈 Resultados Clave
Identificación de segmentos de usuarios: Mensajeros Intensivos, Llamadores Intensivos
Clasificación por grupos etarios: Joven, Adulto, Adulto Mayor
Detección de comportamientos atípicos en el uso de servicios
Recomendaciones para optimización de planes comerciales

🔍 Principal Hallazgo
Usuarios "Llamadores Intensivos" - Oportunidad de Planes Personalizados

Identificamos un segmento de 109 usuarios (2.7% del total) con comportamiento atípico en minutos de llamada, divididos en dos grupos:

Grupo 1: 60-100 minutos mensuales (usuarios de llamadas largas)
Grupo 2: 120-180 minutos mensuales (usuarios de llamadas muy largas)
Implicación comercial: Estos usuarios representan una oportunidad clara para:

Crear planes premium especializados en llamadas
Implementar beneficios personalizados para retener a estos usuarios de alto valor
Desarrollar estrategias de upselling dirigidas
Este patrón sugiere que existe demanda no satisfecha para planes con mayor capacidad de minutos, lo que podría traducirse en mayor rentabilidad por usuario y mejor satisfacción del cliente.
